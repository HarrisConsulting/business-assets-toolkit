# Color Palette

Complete HarrisConsulting color palette with hex codes, RGB values, and usage guidance.

---

## 🎨 Primary Palette

### Brand Primary — Deep Navy
- **Hex:** `#1A3C5E`
- **RGB:** `26, 60, 94`
- **Use for:** Primary headings, navigation backgrounds, main CTA buttons, document headers

### Brand Secondary — Sky Blue
- **Hex:** `#2A7ABF`
- **RGB:** `42, 122, 191`
- **Use for:** Links, accent borders, secondary buttons, icon highlights

### Brand Accent — Amber
- **Hex:** `#F4A620`
- **RGB:** `244, 166, 32`
- **Use for:** Call-to-action highlights, badges, key data callouts, pricing highlights

---

## 🩶 Neutral Palette

### Dark — Near Black
- **Hex:** `#1C1C1C`
- **RGB:** `28, 28, 28`
- **Use for:** Primary body text, dark mode backgrounds

### Charcoal
- **Hex:** `#374151`
- **RGB:** `55, 65, 81`
- **Use for:** Secondary headings, dark surface UI

### Mid Gray
- **Hex:** `#6B7280`
- **RGB:** `107, 114, 128`
- **Use for:** Captions, helper text, disabled states

### Light Gray
- **Hex:** `#F3F4F6`
- **RGB:** `243, 244, 246`
- **Use for:** Page backgrounds, card fills, dividers

### White
- **Hex:** `#FFFFFF`
- **RGB:** `255, 255, 255`
- **Use for:** Content areas, form fields, clean backgrounds

## ♿ Accessibility

All color combinations used for text must meet **WCAG 2.1 AA** contrast requirements (minimum 4.5:1 for normal text, 3:1 for large text).

### Verified Combinations

| Text Color | Background | Contrast Ratio | WCAG |
|------------|------------|----------------|------|
| `#1C1C1C` | `#FFFFFF` | 18.1:1 | ✅ AAA |
| `#FFFFFF` | `#1A3C5E` | 9.2:1 | ✅ AAA |
| `#FFFFFF` | `#2A7ABF` | 4.7:1 | ✅ AA |
| `#1C1C1C` | `#F3F4F6` | 17.2:1 | ✅ AAA |
| `#1A3C5E` | `#F4A620` | 5.8:1 | ✅ AA |

Use [Accessible Colors](https://accessible-colors.com) or [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) to verify any new combinations.
