# business-assets-toolkit

A curated knowledge base and resources repository for HarrisConsulting business projects. Use this as a **Template Repository** to quickly spin up new client projects pre-loaded with branding guidelines, templates, and tool references.

## 📁 Structure

```
business-assets-toolkit/
├── design-resources.md       # Master list of curated design & asset resources
├── branding/
│   ├── brand-guidelines.md   # Brand identity standards and usage rules
│   └── color-palette.md      # Color palette and typography references
├── templates/
│   ├── project-brief.md      # Client project brief template
│   ├── proposal-template.md  # Consulting proposal template
│   └── issue-templates/      # GitHub issue templates for new projects
└── tools/
    ├── recommended-tools.md  # Curated list of productivity & design tools
    └── onboarding-checklist.md # New project onboarding checklist
```

## 🚀 Using This as a Template

This repository is configured as a **GitHub Template Repository**. To start a new project:

1. Click the **"Use this template"** button at the top of this page
2. Name your new repository and choose visibility
3. Your new repo will include all reference files, templates, and checklists
